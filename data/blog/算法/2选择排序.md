# 选择排序

选择排序有两个模型

## 原数组排序

使用替换的思想，取剩余最大值替换当前值

```php
function selection_sort($arr)
{
    $len = count($arr);

    // 这里$i<len也行，只因最后剩一个不需要排序，所以-1
    for ($i = 0; $i < $len - 1; $i++) {
        $max_index = $i;

        // 找出最大值，注意：$j从$i+1开始
        for ($j = $i + 1; $j < $len; $j++) {
            if ($arr[$j] > $arr[$max_index]) {
                $max_index = $j;
            }
        }
        $tmp = $arr[$i];
        $arr[$i] = $arr[$max_index];
        $arr[$max_index] = $tmp;
    }

    return $arr;
}

$arr = [3, 5, 1, 9, 2, 8, 7];
$ret = selection_sort($arr);
var_export($ret);
```

## 新数组排序

从旧数组依次找出剩余最大值，追加到新数组

```php
function selection_sort($arr)
{
    $len = count($arr);
    $ret = [];

    for ($i = 0; $i < $len; $i++) {
        $max_index = 0;
        $max = 0;

        foreach ($arr as $k => $item) {
            if ($item > $max) {
                $max_index = $k;
                $max = $item;
            }
        }

        $ret[] = $arr[$max_index];
        unset($arr[$max_index]);
    }

    return $ret;
}

$arr = [3, 5, 1, 9, 2, 8, 7];
$ret = selection_sort($arr);
var_export($ret);
```
